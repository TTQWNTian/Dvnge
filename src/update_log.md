### 使用方法：
**打字音效：** 在章节数据节点中添加 `打字音效` 字段，配置音效路径、音量以及播放间隔（第1个字符必播，之后每隔 N 个字符播放一次）。

```javascript
打字音效: {
    启用: true,
    路径: "音效/打字声.mp3",
    音量: 0.5,// 音量 0-1
    间隔字符数: 3// 每几个字符播放一次
}
```
`打字音效`同`立绘`、`背景`等一样，会沿用上一个节点的配置。
若需关闭打字音效，可设为`null`：
```javascript
打字音效: null
```   

**角色样式配置：** 此功能是给不同角色名称设置不同样式的功能，比如角色是 **“AAA”** 时文本颜色是`red`， **“BBB”** 时颜色是`green`什么的。使用方法：
```javascript
// 找到js/样式配置.js文件

// 角色样式配置
const 角色样式表 = {
    // ''中的是角色名字，后面的{}中设置样式代码
    'AAA': { color: 'red'},
    'BBB': { color: 'green', fontSize: '1.2em' },// 也可以修改除color之外的任何样式
};
```   

**&lt;wait&gt;：** 此标签功能是在逐字显示到这个段落时停顿n时间再继续逐字显示文本，格式：`<wait=数字>`或 `<wait>`（默认1秒）。
```javascript
内容: "这样吗……<wait=0.5>我明白了。",
// 会在“……”显示完之后停顿0.5秒再开始显示“我明白了。”
```

### 注意事项
修复章节加载稳定性之后添加章节写发有变，现在只需要在`TTQ视觉小说引擎.js`中添加修改。
```javascript
const 章节库 = {};// 本次修改后此章节库不再需要写任何东西，留空即可

// 章节文件列表_在这个列表添加修改/章节
const 章节文件列表 = [
    { 名称: "序章", 路径: "js/章节/序章.js" },
    // 后续添加更多章节：{ 名称: "第一章", 路径: "js/章节/第一章.js" },
];
```
因为之前写在`章节库`的章节信息要全部重新用新的格式添加在`章节文件列表`中，以及需要把每个章节前面的`const`换成`var`，所以没有需求不建议更新。
