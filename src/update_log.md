# Dvnge 更新日志

## v1.7.0-dev
### 新增功能
- 打字音效

**使用方法：**
在章节数据节点中添加 `打字音效` 字段，配置音效路径、音量以及播放间隔（第1个字符必播，之后每隔 N 个字符播放一次）。

```javascript
打字音效: {
    启用: true,
    路径: "音效/打字声.mp3",
    音量: 0.5,// 音量 0-1
    间隔字符数: 3// 每几个字符播放一次
}
```
`打字音效`同`立绘`、`背景`等一样，会沿用上一个节点的配置。
若需关闭打字音效，可设为`null`：
  ```javascript
  打字音效: null
  ```
