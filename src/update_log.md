### 多语言使用方法

#### 1. 配置语言列表
在 `js/语言配置.js` 中定义支持的语言：
```js
const 语言配置 = {
    默认语言: 'zh',
    语言列表: {
        'zh': '中文',
        'en': 'English',
        'ja': '日本語'
    }
};
```

#### 2. 界面文字多语言
在 `语言配置.界面文字` 中为每个界面元素添加多语言文本：
```js
界面文字: {
    '快进': { 'zh': '快进', 'en': 'Skip', 'ja': 'スキップ' },
    '隐藏': { 'zh': '隐藏', 'en': 'Hide', 'ja': '隠す' }
}
```

#### 3. 章节对话多语言
在章节数据中为角色和内容添加多语言字段：
```js
{
    角色: '中文角色名',
    角色_en: 'English Name',
    角色_ja: '日本語名',

    内容: '中文内容',
    内容_en: 'English Content',
    内容_ja: '日本語内容'
}
```

#### 4. 选项多语言
为选项按钮添加多语言文本字段：
```js
{
    文本: '中文选项',
    文本_en: 'English Option',
    文本_ja: '日本語選択肢'
}
```

#### 5. 输入组件多语言
为输入节点添加多语言提示文字：
```js
{
    提示文字: '请输入名字',
    提示文字_en: 'Please enter your name',
    提示文字_ja: '名前を入力してください',

    占位符: '你的名字',
    占位符_en: 'Your name',
    占位符_ja: 'あなたの名前',

    按钮文字: '确认',
    按钮文字_en: 'Confirm',
    按钮文字_ja: '確認'
}
```
**完成！**

#### 语言切换方式
- **开始界面切换**：点击开始界面的「语言」按钮，在弹层中选择语言
- **自动记忆**：切换后的语言偏好自动保存到 `localStorage`，下次打开时自动恢复
- **运行时切换**：在游戏内通过调用 `切换语言('语言代码')` 函数动态切换

#### 注意事项
- 默认语言的内容使用基础字段名（如 `内容`、`角色`），无需加后缀
- 如果某语言字段缺失，引擎会自动回退到默认语言的内容
- 语言代码必须与 `语言列表` 中定义的键名一致
