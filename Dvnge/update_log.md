- 颜色和图片背景新增切换时淡入淡出效果。
- 背景视频以及立绘视频新增自定义封面（要在 `TTQ视觉小说引擎.js` 里面换成你的路径）
- 新增 `自定义功能` 节点，更方便的设置使用自定义功能。

**自定义功能 使用方法**
```javascript
{
    自定义功能: `
        // 直接执行代码：
        alert('提示');
        // 或者调用自定义功能.js文件中的函数：
        提示('hello');
    `
}
```
后一种方法在 `自定义功能.js` 文件中这样写：
```javascript
function 提示(消息) {
    alert(消息);
};
```
