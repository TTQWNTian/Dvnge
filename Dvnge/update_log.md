# Dvnge_v1.3.2
**新增**：背景与立绘可设置视频格式   
**新增**：设置视频的几个配置节点   

**使用方法**：
```JavaScript
// 背景是视频
{
    背景: "视频.mp4",
    背景媒体: {
        循环: true, // 也可设置为false，设置为false后面的几个配置就都不需要了，如果不设置循环与否会默认一直循环
        播放次数: -1, // -1表示无限循环，其他正整数为具体播放次数
        播放间隔: 0, // 毫秒
        音量: 1 // 0-1之间的数值
    }
}
// 立绘是视频
{
    中立绘: {,
        路径: "视频.mp4",
        媒体: {// 除了这个节点名称和位置不一样，其他同背景媒体设置
            循环: true,
            播放次数: -1,
            播放间隔: 0,
            音量: 1
        }
    },
}
```
